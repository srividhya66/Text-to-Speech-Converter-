# 🗣️ Text-to-Speech Converter (Telugu Support)

A simple **Text-to-Speech Converter** built with **HTML, CSS, JavaScript (Web Speech API)**.  
Supports **Telugu voice (te-IN)** along with other languages. 🎙️
